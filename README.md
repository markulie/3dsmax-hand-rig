# Free Hand Rig for 3ds Max (2015)

A free human hand rig created for **Autodesk 3ds Max**.

This rig was originally made in **2015** as a personal project and is now released as a free download. It includes a skinned hand with an animation-friendly bone setup suitable for posing, rendering, learning, or as a starting point for your own projects.

## Features

- Human hand rig
- Skin already bound to the skeleton
- Individual finger controls through bone hierarchy
- Lightweight and easy to modify
- Suitable for learning rigging and animation
- Free to use

---

## Software

- Autodesk 3ds Max
- Created in **2015**

> Depending on your version of 3ds Max, you may need to resave or update the scene.

---

## Contents

- Hand mesh
- Skeleton (bones)
- Skin modifier
- Ready for posing and animation
